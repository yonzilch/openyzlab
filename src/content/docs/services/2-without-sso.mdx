## 状态监控

### Komari
- **URL**: https://monitor.yzlab.eu.org
- **用途**: 服务器状态监控仪表板
- **使用场景**: 实时监控系统资源与服务健康状态
- **主要功能**: 实时 CPU / 内存 / 网络状态展示、历史趋势、告警通知

### Gatus
- **URL**: https://status.yzlab.eu.org
- **用途**: 应用服务状态监控仪表板
- **使用场景**: 定时监测应用可用情况
- **主要功能**: 应用服务状态展示、历史记录、SLA 查询

## 文件与存储

### OpenList
- **URL**: https://share.yzlab.eu.org
- **用途**: 多存储聚合文件列表服务
- **使用场景**: 跨存储文件浏览与分享、WebDAV 挂载
- **主要功能**: 多存储后端支持、在线预览、分享链接生成
### File-Transfer-Go
- **URL**: https://transfer.yzlab.eu.org
- **用途**: 一次性临时文件传输工具
- **使用场景**: 大文件快速分享、临时单次下载链接
- **主要功能**: 过期自动删除、简单上传界面、无需注册

### Microbin
- **URL**: https://bin.yzlab.eu.org
- **用途**: 轻量级代码与文本片段分享平台
- **使用场景**: 代码粘贴分享、临时文本存储
- **主要功能**: 语法高亮、自定义过期时间、密码保护

## 搜索引擎

### Twonav
- **URL**: https://nav.yzlab.eu.org
- **用途**: 导航页（可认为是特殊的搜索引擎）
- **使用场景**: 找寻互联网各类工具、资源、信息
- **主要功能**: 自定义分类书签、内置搜索、响应式设计

### SearXNG
- **URL**: https://searxng.yzlab.eu.org
- **用途**: 元搜索引擎
- **使用场景**: 匿名搜索、聚合多引擎结果
- **主要功能**: 无追踪、多引擎并行查询、结果分类过滤

### Luxirty Search
- **URL**: https://search.yzlab.eu.org
- **用途**: 搜索互联网 (Search the Internet)
- **使用场景**: 替代 Google 搜索
- **主要功能**: 屏蔽内容农场，无广告，无跟踪，干净，简洁，快

## 实用工具

### BentoPDF
- **URL**: https://pdf.yzlab.eu.org
- **用途**: 在线 PDF 处理工具集
- **使用场景**: PDF 合并、拆分、格式转换、页面操作
- **主要功能**: 多种 PDF 操作、OCR 文字识别、格式互转

### IT-Tools
- **URL**: https://it-tools.yzlab.eu.org
- **用途**: 面向开发者与 IT 人员的在线工具箱
- **使用场景**: 日常编码转换、数据格式化、加解密操作
- **主要功能**: 100+ 种工具、支持离线使用、覆盖编码 / 加密 / 网络 / 时间等类别

### Omni-Tools
- **URL**: https://tool.yzlab.eu.org
- **用途**: 多功能通用在线工具集
- **使用场景**: 文本处理、数据格式转换、日常效率工具
- **主要功能**: 多类型实用工具、简洁界面、快速访问

### CyberChef
- **URL**: https://cyberchef.yzlab.eu.org
- **用途**: 数据分析与编解密「网络瑞士军刀」
- **使用场景**: 编码解码、加密解密、二进制 / 文本格式转换与分析
- **主要功能**: 可视化拖拽「配方」、数百种算法、支持链式组合操作

### AriaNG
- **URL**: https://ariang.yzlab.eu.org
- **用途**: Aria2 的现代化 Web 管理界面
- **使用场景**: 远程管理 Aria2 下载任务
- **主要功能**: HTTP / BitTorrent / Magnet 多协议支持、任务队列管理、限速控制

### Vert
- **URL**: https://vert.yzlab.eu.org
- **用途**: 本地优先的文件格式转换工具
- **使用场景**: 在浏览器中转换音频、视频、图片等格式，文件不上传服务器
- **主要功能**: 多格式支持（音频 / 视频 / 图片 / 文档）、本地处理保护隐私、无需安装

### Etherpad
- **URL**: https://pad.yzlab.eu.org
- **用途**: 极简实时协作文本编辑器
- **使用场景**: 快速协作草稿、临时记事、头脑风暴
- **主要功能**: 毫秒级实时同步、修订历史、轻量无障碍访问
