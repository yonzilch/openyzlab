## 开发工具

### Forgejo
- **URL**: https://git.yzlab.eu.org
- **用途**: 轻量级 Git 仓库托管平台，Go 语言编写的 Gitea 社区分支
- **使用场景**: 代码版本管理、团队协作开发、CI/CD 流水线集成
- **主要功能**: Git 仓库管理、Issues 追踪、Pull Requests 审查、Actions 工作流

### Woodpecker CI
- **URL**: https://ci.yzlab.eu.org
- **用途**: 开源自托管 CI/CD 引擎，与 Forgejo 深度集成
- **使用场景**: 代码提交后自动化构建、测试与部署
- **主要功能**: Docker 原生流水线、插件生态、多仓库并发执行

### Yaade
- **URL**: https://api.yzlab.eu.org
- **用途**: 自托管 API 开发与调试平台
- **使用场景**: 接口联调测试、团队共享 API 集合、多环境切换
- **主要功能**: 可视化请求构建、环境变量管理、响应分析、团队协作

### OpenGist
- **URL**: https://gist.yzlab.eu.org
- **用途**: 自托管代码片段分享服务，类似 GitHub Gist
- **使用场景**: 代码片段保存与分享、技术笔记存储
- **主要功能**: Git 版本控制、Markdown 渲染、公开 / 私密模式

## 安全与认证

### Vaultwarden
- **URL**: https://vault.yzlab.eu.org
- **用途**: Bitwarden 兼容的自托管密码管理服务端
- **使用场景**: 个人与团队密码安全存储与跨设备同步
- **主要功能**: 密码生成与存储、浏览器自动填充、跨设备同步、组织共享

### VoidAuth
- **URL**: https://auth.yzlab.eu.org
- **用途**: 轻量级统一身份认证服务，为本站各服务提供 SSO
- **使用场景**: 单点登录管理、OAuth 应用授权、用户账号管理
- **主要功能**: OpenID Connect / OAuth 2.0、多应用接入、用户与权限管理

## 社交与通讯

- **URL**: https://social.yzlab.eu.org

- **主要功能**: Mastodon API 兼容、极低资源占用、自定义可见性

#### Elk <small>（GoToSocial / Mastodon 前端）</small>

- **使用场景**: 浏览联邦时间线、发帖互动
- **主要功能**: 流畅的现代化界面、多账号切换、通知管理、键盘快捷键

#### Masto-FE <small>（GoToSocial / Mastodon 前端）</small>
- **URL**: https://masto-fe.yzlab.eu.org
- **用途**: Mastodon 官方 Web 前端
- **使用场景**: 通过标准 Mastodon 界面访问联邦社交实例
- **主要功能**: 时间线浏览、发帖与互动、用户与列表管理
